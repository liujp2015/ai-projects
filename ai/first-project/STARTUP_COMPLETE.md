## 🚀 首次使用指南

### 步骤1: 登录后台管理

1. 打开浏览器访问 http://localhost:8000/login
2. 输入账号密码登录
3. 登录成功后进入仪表盘

### 步骤2: 创建基础数据

#### 2.1 创建品牌
- 点击左侧菜单"品牌管理"
- 点击"添加品牌"
- 填写品牌信息并保存

#### 2.2 创建分类
- 点击左侧菜单"分类管理"
- 点击"添加分类"
- 填写分类信息并保存

#### 2.3 创建商品
- 点击左侧菜单"商品管理"
- 点击"添加商品"
- 填写商品信息：
  - 标题、描述
  - 原价、折扣价
  - **跳转链接**（重要！）
  - 选择品牌和分类
  - 购物平台（淘宝、京东等）
  - 商品图片URL
- 保存

#### 2.4 创建Banner
- 点击左侧菜单"Banner管理"
- 点击"添加Banner"
- 填写Banner信息并保存

### 步骤3: 查看前台效果

1. 访问 http://localhost:3001
2. 应该能看到：
   - Banner轮播图
   - 商品列表
3. 点击商品查看详情
4. 测试"去购买"按钮

### 步骤4: 测试用户功能

1. 在前台注册新用户
2. 登录后查看个人中心
3. 查看订单列表（初始为空）

## ⚠️ 重要提示

### 1. 服务状态

如果服务未正常启动，检查：
- 终端是否有错误信息
- 端口是否被占用
- 数据库连接是否正常
- Redis服务是否运行

### 2. 环境变量

确保 `backend/.env` 文件已正确配置：
```env
DATABASE_URL="postgresql://postgres:123456@localhost:5432/rebate_db"
REDIS_HOST=localhost
REDIS_PORT=6379
```

### 3. Redis服务

确保Redis服务正在运行（用于验证码和队列）：
```bash
redis-cli ping
# 应该返回: PONG
```

### 4. 数据库

确保PostgreSQL服务正在运行，数据库 `rebate_db` 已创建。

## 🛠️ 服务管理

### 查看服务状态

在终端中可以看到服务的运行日志。

### 停止服务

在对应的终端按 `Ctrl+C` 停止服务。

### 重启服务

1. 停止服务（Ctrl+C）
2. 重新运行启动命令：
   ```bash
   # 后端
   cd backend && npm run start:dev
   
   # 前台
   cd frontend && npm run dev
   
   # 后台管理
   cd admin && npm start
   ```
