## 🎯 首次使用流程

### 1. 登录后台管理

访问 http://localhost:8000/login，使用默认账号登录。

### 2. 创建基础数据

#### 创建品牌
1. 点击左侧菜单"品牌管理"
2. 点击"添加品牌"按钮
3. 填写品牌信息（名称、描述等）
4. 保存

#### 创建分类
1. 点击左侧菜单"分类管理"
2. 点击"添加分类"按钮
3. 填写分类信息
4. 保存

#### 创建商品
1. 点击左侧菜单"商品管理"
2. 点击"添加商品"按钮
3. 填写商品信息：
   - 标题
   - 原价、折扣价
   - 跳转链接（重要！）
   - 选择品牌和分类
   - 购物平台
   - 商品图片URL
4. 保存

#### 创建Banner
1. 点击左侧菜单"Banner管理"
2. 点击"添加Banner"按钮
3. 填写Banner信息（图片URL、链接等）
4. 保存

### 3. 查看前台效果

1. 访问前台首页 http://localhost:3001
2. 应该能看到：
   - Banner轮播图
   - 商品列表
3. 点击商品查看详情
4. 测试"去购买"按钮

### 4. 测试用户功能

1. 在前台注册新用户
2. 登录后查看个人中心
3. 查看订单列表

## ⚠️ 常见问题

### 问题1: 后端服务无法启动

**可能原因:**
- 数据库连接失败
- Redis连接失败
- 端口被占用

**解决方案:**
1. 检查 `backend/.env` 文件配置
2. 确保PostgreSQL服务运行
3. 确保Redis服务运行
4. 检查端口3000是否被占用

### 问题2: 前台无法访问

**可能原因:**
- 端口被占用
- 依赖未安装

**解决方案:**
1. 检查端口3001是否被占用
2. 重新安装依赖: `cd frontend && npm install`

### 问题3: 后台管理无法访问

**可能原因:**
- 端口被占用
- UmiJS配置问题

**解决方案:**
1. 检查端口8000是否被占用
2. 查看终端错误信息

### 问题4: 登录失败

**可能原因:**
- 后端服务未启动
- 数据库连接问题
- 账号密码错误

**解决方案:**
1. 确认后端服务正在运行
2. 检查数据库连接
3. 确认使用正确的账号: admin@example.com / admin123
