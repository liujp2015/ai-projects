# 返利折扣网前台

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Zustand
- Socket.io Client

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:3001 启动。

## 项目结构

```
src/
├── app/            # Next.js App Router
├── components/     # 组件
├── lib/            # 工具库
└── store/          # 状态管理
```
